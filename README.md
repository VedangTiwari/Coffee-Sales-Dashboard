# ☕ Coffee Sales Dashboard

An interactive Excel dashboard built to analyze sales performance across coffee types, customer segments, and geographies.

---

## 📊 Project Overview

This project demonstrates how raw transactional data can be transformed into a structured, interactive MIS dashboard using Microsoft Excel.

---

## 🔧 What This Project Covers

- Data cleaning and structuring across 3 related tables — **orders, customers, products**
- Use of **XLOOKUP** and **INDEX MATCH** to dynamically combine data across sheets
- Creation of **Pivot Tables** to summarize:
  - Sales by country  
  - Sales by coffee type  
  - Customer-wise revenue  
- KPI cards displaying:
  - **Total Sales**: $45,134  
  - **Total Orders**: 1,000  
  - **Average Order Value**: $45.13  
- Interactive **slicers** for:
  - Roast Type  
  - Coffee Type  
  - Size  
  - Loyalty Card  
- **Timeline slicer** for date-based filtering (2019–2022)
- **Line chart** to track sales trends over time
- Identification of **Top 5 customers by revenue**

---

## 🛠 Tools & Features Used

- Microsoft Excel  
  - Pivot Tables  
  - XLOOKUP  
  - INDEX MATCH  
  - Slicers & Timeline  
  - Charts & KPI Cards  

---

## 📈 Key Business Questions Answered

- Which country generates the most revenue?  
- Which coffee type performs best over time?  
- Who are the top spending customers?  
- How does loyalty card usage impact purchasing behavior?  

---

## 📌 Key Learnings

- Building end-to-end dashboards from raw data  
- Creating dynamic reports using Pivot Tables and slicers  
- Designing KPI-driven dashboards for business decision-making  
- Improving data visualization and storytelling
