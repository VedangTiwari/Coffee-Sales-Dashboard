Coffee Sales Dashboard
An interactive Excel dashboard built to analyze sales performance across coffee types, customer segments, and geographies.
What this project covers

Data cleaning and structuring across 3 related tables — orders, customers, products
XLOOKUP and INDEX MATCH formulas to pull data across sheets dynamically
Pivot Tables summarizing sales by country, coffee type, and customer
KPI cards showing Total Sales ($45,134), Total Orders (1,000), and Avg Order Value ($45.13)
Interactive slicers for Roast Type, Coffee Type, Size, and Loyalty Card
Timeline slicer for date-based filtering across all charts
Line chart tracking sales trends over time (2019–2022)
Top 5 customers by revenue

Tools used

Microsoft Excel — Pivot Tables, XLOOKUP, INDEX MATCH, slicers, timeline, charts

Key business questions answered

Which country generates the most revenue?
Which coffee type sells best over time?
Who are the top spending customers?
How does loyalty card usage affect purchasing?
